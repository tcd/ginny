## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ginny'
```

And then execute:

```ruby
bundle
```

Or install it yourself as:

```ruby
gem install ginny
```

## Development

After checking out the repo, run `bin/setup` to install dependencies. 
Then, run `rake test` to run the tests.
